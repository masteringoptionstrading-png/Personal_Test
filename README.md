
# Trading Performance Dashboard

A professional static trading dashboard built with:

- `index.html`
- `styles.css`
- `script.js`

## Features

- KPI cards
- Equity curve
- Daily P/L chart
- Win vs Loss chart
- P/L % per trade chart
- Full trade history table

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repo.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / root
5. Save.
6. GitHub will give you a public URL for the dashboard.

## Update your trade data

Open `script.js` and replace the `tradeData` array with your latest trades.

## Notes

- This is a read-only public webpage once published on GitHub Pages.
- No backend required.
- Chart.js is loaded from a CDN.
